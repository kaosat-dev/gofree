## Usage

This package can currently only be used in the new version of JSCAD web V2 (written as npm module)
  - clone this repository
  - either
    * install the web ui for JSCAD V2 (V2 branch)
    * go to jscad.xyz

- drag & drop this folder onto the JSCAD UI
